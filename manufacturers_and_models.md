## Research Strategy

1. **Visit Manufacturer Websites**: Navigate to specific model pages for detailed specifications
2. **Contact Dealers**: Use dealer locators to get current pricing and availability
3. **Request Brochures**: Most manufacturers offer downloadable spec sheets
4. **Visit Dealerships**: For hands-on inspection and detailed specifications
5. **Check Online Marketplaces**:
   - caravancampingsales.com.au
   - caravansales.com.au
   - tradingpost.com.au

## Key Specifications to Gather

For each model, collect:
- ✓ External dimensions (L x W x H)
- ✓ Internal dimensions
- ✓ Tare weight
- ✓ ATM (Aggregate Trailer Mass)
- ✓ Ball weight
- ✓ Axle configuration (single/twin)
- ✓ Solar panel wattage
- ✓ Battery capacity (Ah and type - AGM/Lithium)
- ✓ Main bed dimensions
- ✓ Bunk bed dimensions and quantity
- ✓ Water tank capacity
- ✓ Price (RRP and any current promotions)
- ✓ Roof type (confirm hard top)
- ✓ Suspension type
